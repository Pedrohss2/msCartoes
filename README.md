## Ms cartoes 
Uma API que permite criar um cliente

## Funcionalidades

- CRUD cliente

## Uso/Exemplos
End-point para criar um novo vendedor

Method: POST
```
http://localhost:8080/vendedor
```
Body (raw)
```
{
    "nome": "teste teste",
    "bandeira": "VISA",
    "renda": 5000.0,
    "limiteBasico": 20000.0
}
```
